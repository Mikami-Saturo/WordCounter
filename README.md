# WordCounter

Programme Java de comptage de mots dans un fichier texte.
Lit un fichier texte ligne par ligne, sépare les mots
par les espaces et affiche le nombre total de mots.

## Fonctionnalités
- Lecture d'un fichier texte
- Comptage des mots ligne par ligne
- Affichage du nombre total de mots

## Comment exécuter
1. Cloner le dépôt
2. Ouvrir le projet dans IntelliJ
3. Ajouter un fichier texte nommé "texte.txt" à la racine du projet
4. Exécuter la classe Main
